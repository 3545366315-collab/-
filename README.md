
今天我和炒股搭子聊天，为我们卖飞的一支又一支股感到懊悔。身为两个短线交易者，我们的选股眼光其实都不太差，但是收益率却波动很大，迟迟不见起色。后面我们意识到了，这恰恰是短线的系统性缺陷。
对于那些天赋机缘努力时机都集于一身的宠儿，他们是能够通过短线交易来实现一年十倍甚至一年二十倍的涨幅，这些人天生就是吃短线这口饭的。
但是对于大部分的其他人，想要凭借短线去暴富是很难很难的，可以去追求短线交易，可能会输个底朝天，因为短线对买卖点都有极高的要求，一不小心就会低卖高买。而且短线极易受到情绪的干扰，由短线起伏而产生的情绪又会反过来影响生活，所以短线很难，超短更难。

所以我觉得投资的要义不是快进快出的投机，而是长期持有的智慧。找到一支好的股票，然后长期持有，接受它的横盘与一段时间的阴跌，然后才能得到恰当的时机的爆发式增长，进而获得更大的利润。每年都吃这么一波两波，相信复利能够给你带来超乎想象的收益。

去年我在中国人民大学上了肖刚老师的《财富管理与幸福》课程，老师用一张图给我们展示了复利的智慧。从25岁开始每年定投5万元，按照每年10%的收益来算的话，67岁就可以赚到3000万。我就根据肖刚老师的想法，把这个复利图做成了一个交互网站，可以自由的选择定投的金额、预期的收益率、开始定投的时间与结束的时间，然后就可以看到在复利这个奇迹的帮助下，自己未来可以赚到的钱，以此来激励自己做一个耐心的投资者。
